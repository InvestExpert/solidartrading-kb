# SolidarTrading — план доработки базы знаний

## [ИТОГ]

Текущая база уже пригодна для тестового ИИ-ассистента.

Но для нормальной продажной работы не хватает не общей философии, а конкретных файлов по ключевым вопросам:

1. условия подключения;
2. риски и 4 степени защиты;
3. полная партнёрская программа;
4. FAQ и возражения;
5. материалы и ссылки;
6. визуальные материалы для бота.

Главная цель доработки: чтобы бот мог уверенно отвечать на вопросы клиента и партнёра без выдумывания, без сухого дисклеймера и без потери продажной подачи.
