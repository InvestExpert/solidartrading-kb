# ИСТОЧНИКИ трафика для АРБИТРАЖА Трафика!!! | Арбитраж Трафика с Нуля

## Ссылка

- Видео: https://www.youtube.com/watch?v=UyljNNG49iQ

## Кратко

Видео даёт базовую классификацию источников трафика для арбитража: условно-бесплатные источники, платные рекламные сети, основные рекламные форматы и модели оплаты.

Материал полезен не как источник уникальных площадок, а как обучающая схема для раздела `TiTan Bots - Трафик`: как делить источники, почему платный трафик удобнее для быстрых тестов, какие форматы учитывать в карте источников и какие метрики считать.

## Основная логика видео

Автор делит трафик на две группы:

1. **Условно-бесплатный трафик** — не оплачивается напрямую деньгами за клик/показ, но требует времени, контента, активности, расходников или личного присутствия.
2. **Платный трафик** — покупается через рекламные сети и позволяет быстрее тестировать связки, управлять бюджетом и масштабировать рабочие кампании.

Ключевой тезис: в арбитраже основной интерес чаще представляет именно платный трафик, потому что он управляемый, измеримый и позволяет быстро проверять гипотезы.

## Форматы платной рекламы

### Таргетированная реклама

Используется в социальных сетях. Сильная сторона — возможность выбирать аудиторию по интересам, поведению, демографии и другим признакам.

**Применимость для базы:**
- добавить в группу `Социальные рекламные кабинеты`;
- учитывать как управляемый paid-source;
- для финансовых и спорных офферов обязательно проверять модерацию.

### Контекстная реклама

Показывает объявления на основе поисковых запросов и контекста страниц.

Главные площадки:
- Google Ads;
- Яндекс Директ.

**Применимость для базы:**
- добавить в группу `Поисковая реклама`;
- фиксировать как источник intent-трафика;
- учитывать высокую конкуренцию и дорогой клик в финансовых тематиках.

### Push-уведомления

Массовый формат уведомлений на мобильных устройствах и десктопах.

**Применимость для базы:**
- группа `Push и pop-сети`;
- подходит для дешёвого объёма;
- требует отдельной проверки качества.

### Тизерная реклама

Интригующая картинка + короткий текст.

**Применимость для базы:**
- группа `Тизерные сети`;
- может работать в дешёвых и массовых вертикалях;
- для белых офферов использовать осторожно.

### Нативная реклама

Похожа на тизер, но органично вписывается в сайт и выглядит более аккуратно.

**Применимость для базы:**
- группа `Тизерные и native-рекламные сети`;
- лучше подходит для объясняющих и “мягких” посадочных страниц.

### Pop-up / Popunder / Clickunder

Агрессивные форматы:
- pop-up открывается поверх сайта;
- popunder открывается позади основного окна;
- clickunder открывается после клика по сайту.

**Применимость для базы:**
- группа `Push / pop / redirect`;
- использовать только как отдельный рискованный тест;
- высокий риск мусорного трафика и раздражения пользователя.

## Модели оплаты

| Модель | Суть | Комментарий |
|---|---|---|
| Flat / fixed placement | Оплата за время размещения | Например, баннер на срок |
| CPV | Оплата за просмотр | Подходит для видео/просмотров |
| CPC | Оплата за клик | Базовая модель для тестов |
| CPM | Оплата за 1000 показов | В транскрипте сказано “1000 кликов”, но по смыслу CPM = 1000 показов |
| CPA | Оплата за целевое действие | Общая модель result-based оплаты |
| CPL | Оплата за лид | Подходит для регистраций/заявок |
| CPI | Оплата за установку | Для app/install-трафика |
| CPS | Оплата за продажу | Подходит для прямой монетизации |
| Auction | Аукционная ставка | Распределение трафика по bid |

## Практические идеи

### 1. Делить источники не только по площадкам, но и по формату

Один и тот же источник может давать разные типы трафика:

- Facebook = таргет;
- Google = поиск + display;
- Яндекс = поиск + РСЯ;
- Telegram = каналы + чаты + боты + рассылки;
- рекламные сети = push + pop + native + banner.

Для карты источников важно фиксировать не только платформу, но и формат.

### 2. Платный трафик нужен для быстрых тестов

Преимущество платного трафика по видео:

- можно контролировать бюджет;
- можно запускать и останавливать кампании;
- можно быстро проверять связки;
- можно масштабировать рабочие варианты.

Для нашей базы это подтверждает методику: источник должен попадать в тест только через измеримую связку `источник -> аудитория -> посадочная точка -> метрики`.

### 3. ROI — главный принцип арбитража

В видео дана базовая формула:

```text
ROI = (доходы - расходы) / расходы * 100%
```

В расходы надо включать не только рекламный бюджет, но и:
- дизайн;
- переводы;
- хостинг;
- лендинги;
- расходники;
- оплату источника;
- другие технические затраты.

### 4. Выбор источника зависит от оффера и вертикали

В видео выделены критерии:

- разрешён ли оффер правилами источника;
- хватает ли бюджета;
- хватает ли опыта;
- соответствует ли источник вертикали;
- можно ли протестировать связку.

Это можно перенести в `Методику тестирования трафика`.

## Что можно добавить в карту источников

### Новые или уточнённые группы

- `Условно-бесплатный трафик`
- `Платные рекламные сети`
- `Таргетированная реклама`
- `Контекстная реклама`
- `Push-уведомления`
- `Тизерная реклама`
- `Нативная реклама`
- `Pop-up / Popunder / Clickunder`
- `Нестандартный офлайн / ручной трафик`

### Уточнения к существующим группам

- SEO, блоги, форумы и видео-платформы не исключать из общей карты, но помечать как долгий/контентный путь.
- Push, teaser, pop надо явно отделить от более качественных paid-source.
- Facebook Ads в видео назван стартовым источником для новичков, но это мнение автора и требует отдельной проверки под конкретный проект.

## Что можно добавить в реестр источников

Только после отдельного решения:

| Кандидат | Почему |
|---|---|
| Facebook Ads | Автор рекомендует как стартовый платный источник для новичков |
| Push-сети | Высокий объём, но низкое качество |
| Тизерные сети | Массовый дешёвый трафик |
| Нативные сети | Более аккуратный формат тизеров |
| Popunder / Clickunder | Агрессивный, но отдельный paid-format |
| Контекстная реклама | Intent-трафик |

## Что требует теста

### Гипотеза 1

[ГИПОТЕЗА]: платные источники дают более быструю проверку связки, чем условно-бесплатные контентные каналы.

- Вероятность: 90%.
- Как проверить: сравнить время до первых 100 кликов / первых регистраций через paid-source и через органический канал.
- Метрики: время запуска, CPC, CPL, качество регистраций.

### Гипотеза 2

[ГИПОТЕЗА]: push/teaser/pop могут дать дешёвый объём, но качество будет ниже Telegram/поиска/таргета.

- Вероятность: 80%.
- Как проверить: малый тест с одинаковой посадочной точкой и отдельными UTM.
- Метрики: клики, регистрации, валидные регистрации, глубина взаимодействия, жалобы.

### Гипотеза 3

[ГИПОТЕЗА]: Facebook Ads может быть удобным стартовым источником, если оффер проходит модерацию и есть опыт настройки.

- Вероятность: 60%.
- Как проверить: тест с минимальным бюджетом и отдельной посадочной точкой.
- Риск: модерация, блокировки, стоимость расходников, конкуренция.

## Риски

- Условно-бесплатный трафик может требовать больше ресурсов, чем платный тест.
- Тизеры, push и pop часто дают мусорный или низкомотивированный трафик.
- Pop-up/popunder/clickunder могут ухудшать репутацию и качество аудитории.
- Сайты знакомств, AirDrop-ссылки и прямое навязчивое распространение могут быть спамом.
- Facebook и другие соцкабинеты требуют соблюдения правил модерации.
- Видео обучающее и базовое, без конкретных свежих цен и площадок.
