## Краткое содержание

Видео показывает практический workflow для создания коротких AI-видео:

1. Сначала через любую LLM создаётся промпт для video generation.
2. Сервис Google используется для генерации короткого ролика по тексту.
3. Отдельно показывается сценарий image-to-video: сначала создаётся изображение, затем оно анимируется.
4. Для мультяшного ролика предлагается использовать character sheet и storyboard.
5. Storyboard загружается как изображение, а сервису даётся короткая команда следовать правилам из картинки.
6. Итоговый ролик можно подрезать и скачать в MP4.

## Риски / слабые места

### Юридические / platform risks

- Обход ограничений сменой почты — риск нарушения правил платформы.
- Бесплатный доступ, лимиты и доступность сервиса могут измениться.
- Использование чужих лиц, голосов, стилей Disney или узнаваемых брендов может создавать IP / likeness risks.
- Для коммерческого контента нужно отдельно проверять права на использование результата.

### Технические риски

- 8 секунд — жёсткое ограничение формата.
- Качество нестабильно: возможны артефакты персонажа, лишние кадры, смена лица, плохая физика.
- Storyboard не гарантирует покадровое следование.
- Реалистичная физика и сложные сцены могут давать слабый результат.
- Русская речь заявлена как сильная сторона, но нужна проверка на конкретных сценариях.

### Маркетинговые риски

- Вирусный формат сам по себе не равен продаже.
- Нужны метрики: удержание, CTR, переходы, заявки, регистрации.
- Слишком «прикольный» ролик может не объяснять оффер.
- Для финансовых/торговых тем нельзя превращать ролик в вводящую в заблуждение гарантию прибыли.
