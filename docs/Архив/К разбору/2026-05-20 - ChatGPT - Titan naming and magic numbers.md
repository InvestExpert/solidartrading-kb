# Titan naming и magic numbers для TiTan Bots

## Финальные соответствия бот ↔ имя титана

- Barracuda Grid → Atlas
- FLAT → Hyperion
- Bollinger + ADX → Helios
- Chandler Exit → Kronos

## Magic Numbers

### Формула

A1Z26 concatenation

### Примеры

KRONOS:
111815141519

HYPERION:
8251651891514

HYPER:
82516518

## Практические выводы

- MT5 допускает 13-значные magic numbers.
- Длинные значения ухудшают читаемость логов.
- Рекомендуются сокращенные формы magic numbers.

## Архитектура

### Atlas
Grid foundation.

### Hyperion
Hybrid balance system.

### Helios
Trend direction system.

### Kronos
