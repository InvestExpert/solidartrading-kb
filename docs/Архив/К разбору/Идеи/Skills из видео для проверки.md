# Skills из видео для проверки

## Оценка ценности

`1/10`.

Статус: `archive_processed / absorbed`.

Карточка закрыта как отдельный активный список. Полезные кандидаты объединены в каноническую карточку `Skills - идеи развития TiTan Bots`.

## Решение

Не держать два активных backlog по skills.

Эта карточка была источником кандидатов из видео, но в активной очереди она создавала дубль с `Skills - идеи развития TiTan Bots`.

Сохранено в канонической карточке:

- `titan-bots-trading-code-review`;
- `mt5-python-parity-audit`;
- `optimizer-result-analyzer`;
- `third-party-skill-security-audit`;
- `titan-bots-code-version-report`;
- `titan-bots-video-extract`;
- `documentation-to-skill`;
- отложенные frontend/visual/database/MCP направления.

Закрыто без отдельного действия:

- общая идея использовать skills;
- Superpowers как пакет;
- `Редактор без воды`;
- общий Marketing skills;
- общий MCP Builder без конкретной внешней системы;
- поиск готовых skills без выбранной рабочей боли;
- перевод больших документов как отдельный TiTan skill.
