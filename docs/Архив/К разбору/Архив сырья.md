# Архив

Сюда попадают сырые файлы после успешной обработки и переноса полезных знаний в `Вики`.

## Правило

Файл можно перенести в архив только после того, как:

- пользователь одобрил план импорта;
- нужные страницы в `Вики` созданы или обновлены;
- обновлены `Вики/Индекс.md`, `Вики/Источники.md` и `Вики/Журнал.md`.

## Срок хранения

Рабочий срок: 3 месяца.

Рекомендация на старт: 6 месяцев, пока процесс импорта не будет проверен на реальных файлах. После обкатки можно чистить архив от файлов старше 3 месяцев.
