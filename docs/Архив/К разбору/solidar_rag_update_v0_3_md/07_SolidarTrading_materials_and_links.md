# SolidarTrading — материалы, презентации и ссылки

## Назначение

Этот файл нужен, чтобы бот не выдумывал ссылки, не говорил “данных недостаточно” и давал только проверенные материалы из базы.

Важное правило:

```text
Если ссылка есть в этом файле, бот должен её давать. Нельзя отвечать, что видеоинструкция, презентация или PDF-презентация не добавлены в базу, если они указаны ниже.
```

## Презентация проекта

Презентация проекта есть в двух форматах:

1. Видео-презентация.
2. PDF-презентация.

Если пользователь спрашивает:

- есть ли презентация;
- дай презентацию;
- где посмотреть презентацию;
- есть ли PDF;
- есть ли видео о проекте;
- можно материалы проекта;
- хочу посмотреть подробнее;
- что можно отправить человеку;
- есть ли файл презентации;
- можно скачать презентацию;

бот должен дать обе ссылки: видео и PDF.

Правильный ответ:

```text
Да, презентация проекта есть в двух форматах:

Видео-презентация:
https://youtu.be/a-Q_-h5ufzw

PDF-презентация:
https://drive.google.com/file/d/1vX4BxHzmqgOaE-sIMpshwW8ZRVlRSZmv/view?usp=sharing
```

Нельзя отвечать:

```text
Недостаточно данных для надёжного ответа о наличии презентации.
```

Это неверно. Презентация проекта есть: и видео, и PDF.
