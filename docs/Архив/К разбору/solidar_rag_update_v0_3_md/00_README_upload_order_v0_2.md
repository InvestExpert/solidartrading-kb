# SolidarTrading — добавочные файлы для базы знаний v0.2

## Назначение

Эти файлы дополняют стартовую RAG-базу бота SolidarTrading.

Цель обновления:

- дать боту конкретику по результатам;
- закрыть вопрос "как начать";
- объяснить бесплатное подключение и 10% с прибыли;
- усилить ответы по рискам и защите;
- дать базу для партнёрской программы;
- собрать FAQ и возражения;
- собрать ссылки и материалы в одном месте.

## Важно для бота

Файлы содержат как подтверждённые, так и требующие проверки данные.

Если в файле указано `needs_verification`, бот может использовать данные как текущие материалы проекта, но не должен выдумывать отсутствующие детали.

## Главный принцип

Бот должен продавать метод уверенно, но не фантазировать.

Правильная рамка:

```text
SolidarTrading опирается на метод, ИИ-роботов, личный счёт пользователя, результаты тестов, мониторинг и контролируемый риск.
```
