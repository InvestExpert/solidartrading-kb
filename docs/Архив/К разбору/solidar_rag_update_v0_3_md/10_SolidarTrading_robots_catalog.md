## Краткая суть

TiTan Bots — это не один скрипт и не один случайный торговый робот.

Это линейка торговых роботов и алгоритмических систем, которые получают имена титанов:

- **TiTan ATLAS**;
- **TiTan HYPERION**;
- **TiTan HELIOS**;
- **TiTan KRONOS**.

Но для текущего публичного направления **SolidarTrading** основной и первый робот — **TiTan ATLAS**.

Главная логика:

```text
У нас не один робот “на все случаи жизни”, а линейка торговых систем TiTan Bots. Но запускать их планируется постепенно. Сейчас основной фокус — TiTan ATLAS: самый важный, самый проверенный и самый подходящий для долгосрочного публичного подключения через SolidarTrading.
```

## Модель доступа к роботам

В публичном направлении SolidarTrading роботы не продаются пользователю как отдельный товар.

Правильная рамка:

```text
Пользователь не покупает робота как программу. Он подключается к системе через доступную модель подключения: бесплатно или по условиям копитрейдинга, которые описываются в отдельном разделе базы.
```

Важно:

- не позиционировать роботов как обычный софт для покупки;
- не говорить, что пользователь покупает робота;
- не уводить разговор в продажу отдельных роботов;
- для пользователя основной сценарий — подключение к системе, а не покупка исходного кода или программы;
- отдельные роботы могут предоставляться компаниям или партнёрам только по индивидуальному запросу;
- условия подключения, пользования, копитрейдинга и вознаграждения описываются в других разделах базы.

Короткая формулировка:

```text
Мы не продаём роботов обычному пользователю как коробочный софт. В SolidarTrading пользователь подключается к системе и пользуется роботом по условиям подключения или копитрейдинга.
```

Если пользователь спрашивает, можно ли купить робота отдельно:

```text
В текущем публичном направлении роботы не продаются как обычная программа. Основной формат — подключение к системе через SolidarTrading. Отдельные решения для компаний или партнёров возможны только по индивидуальному запросу.
```

## Текущий публичный статус

Важно различать:

1. **Экосистема TiTan Bots** — внутри проекта есть несколько торговых систем и направлений.
2. **SolidarTrading сейчас** — для публичного подключения используется один основной робот: **TiTan ATLAS**.

Правильная рамка:

```text
Внутри TiTan Bots есть линейка роботов, но в SolidarTrading мы не запускаем всё сразу. Сначала выводится самый надёжный и долгосрочный робот — TiTan ATLAS. Остальные решения планируется запускать постепенно, одно за другим, после подготовки, проверки и упаковки.
```

Почему не запускаются все сразу:

- разные роботы решают разные задачи;
- у каждого своя торговая логика;
- каждый робот требует отдельной проверки;
- важно не смешивать риски;
- публичный продукт должен быть понятным для пользователя;
- сначала запускается самый сильный и устойчивый кандидат;
- пользователю проще понять один основной инструмент, чем сразу набор разных алгоритмов.

Коротко:

```text
Роботов много, но публично сейчас главный один — ATLAS. Это не минус, а фильтр качества: сначала даём людям самый проверенный инструмент, а не склад экспериментальных кнопок.
```
