# SolidarTrading — визуальные материалы для бота

## Назначение

Этот файл описывает, какие картинки и карточки нужны для Telegram/Coze-бота SolidarTrading.

Пока картинки могут не выводиться напрямую через обычный Output-блок. Для визуальных экранов лучше использовать chatflow/workflow, если технически подтверждено, что конкретный канал поддерживает картинки.

## Правило для бота

Если картинка технически недоступна, бот должен дать текстовый ответ и ссылку на материал.

## Что нужно проверить технически

- отображаются ли картинки из workflow в Coze Preview;
- отображаются ли картинки в Telegram;
- работают ли карточки в Telegram;
- работают ли кнопки в Telegram;
- если карточки не работают — использовать текстовое меню и цифры.

## Текстовое меню как запасной вариант

```text
Выберите раздел:

1️⃣ Хочу узнать подробнее
2️⃣ Всё о проекте
3️⃣ Стать партнёром
4️⃣ Фишки для партнёров
5️⃣ Спросить ИИ

Напишите цифру или название раздела.
```
