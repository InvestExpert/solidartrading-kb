## Главная идея подключения

SolidarTrading подключается через личный брокерский счёт пользователя.

Пользователь:

- регистрируется в RoboForex;
- использует партнёрский код `xvwag`;
- проходит верификацию;
- создаёт брокерский счёт;
- подключает копирование робота через CopyFX;
- пополняет свой счёт;
- контролирует результат в личном кабинете;
- при необходимости может вывести средства.

Деньги остаются на брокерском счёте пользователя. Пользователь не переводит капитал администратору проекта и не отдаёт деньги в доверительное управление.

Короткая формулировка:

```text
SolidarTrading подключается через личный брокерский счёт. Пользователь регистрируется в RoboForex, проходит верификацию, создаёт счёт, подключает копирование робота и контролирует результат в личном кабинете.
```
