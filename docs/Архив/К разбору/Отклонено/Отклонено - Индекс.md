# Отклонено - Индекс

Архивный список отклоненных идей из слоя `К разбору`.

## Что сюда класть

- идеи, которые рассмотрели и решили не использовать;
- гипотезы, которые не подходят после проверки;
- предложения, которые слишком сложны для текущего этапа;
- подходы, которые уже пробовали и не хотят повторять.

## Зачем хранить отклоненное

Чтобы не возвращаться к одной и той же идее без новой причины.

## Минимальная карточка отказа

- Какая идея рассматривалась.
- Почему ее рассматривали.
- Почему отклонено.
- Когда можно вернуться.
- Связанные страницы.

## Страницы

Пока нет отдельных страниц.
