## Краткое резюме

В чате проводился анализ июньских результатов SmartMode и поведения ADX-фильтра. Основной вывод: ADX-фильтр не должен постоянно ограничивать торговлю. Его задача — активироваться только в реально опасных трендовых фазах рынка.

Также обсуждалась связь VI и ADX как двух разных измерений рынка:
- VI показывает волатильность;
- ADX показывает направленность движения.

Сделан вывод, что лучший режим для сеточной стратегии:
- умеренный VI;
- низкий ADX;
- высокая доля V6;
- минимальные V1–V4.

## Идеи

- Использовать ADX не как постоянный фильтр, а как селективный аварийный ограничитель.
- Комбинировать VI и ADX для более точной оценки режима рынка.
- Разделять:
  - «волатильный флет»;
  - «направленный тренд».

## Гипотезы

- ADX > 28–30 вместе с высоким VI может быть хорошим индикатором опасного тренда для сетки.
- Если месяц уже стабилен по структуре V6/V1, дополнительные фильтры могут ухудшать прибыль без реального снижения риска.
- ADX сам по себе недостаточен без учета VI.

## Выводы

- Хороший месяц для сетки:
  - высокий V6;
  - низкие V1–V4;
  - умеренный VI;
  - ADX ниже ~30.
- Постоянное вмешательство ADX-фильтра может ухудшать результаты.
- SmartMode должен молчать в нормальном рынке и вмешиваться только в кризисных фазах.
- Сильный тренд и высокая волатильность — разные состояния рынка.

## Ошибки и решения

- Ошибка:
  - попытка использовать ADX как универсальный постоянный фильтр.
- Решение:
  - переход к селективному режиму включения только при опасных комбинациях рынка.

- Ошибка:
  - анализ ADX отдельно от структуры событий V1–V6.
- Решение:
  - оценивать фильтр через распределение событий и общую структуру месяца.

## Правила

- Не оценивать фильтры только по прибыли.
- Проверять:
  - распределение V1–V6;
  - DD;
  - структуру месяца;
  - режим рынка.
- Любой фильтр должен:
  - снижать плохие события;
  - не уничтожать V6-прибыль.

## Архитектура

- VI и ADX рассматриваются как независимые оси состояния рынка:
  - VI = интенсивность;
  - ADX = направленность.
- SmartMode должен:
  - минимально вмешиваться;
  - включаться только в опасных режимах;
  - не ломать прибыльный флетовый рынок.
