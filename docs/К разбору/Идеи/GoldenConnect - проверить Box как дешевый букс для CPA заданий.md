# GoldenConnect - проверить Box как дешевый букс для CPA заданий

## Оценка ценности

5/10

## Lifecycle

Это не active/current знание и не ответ для RAG. Идея становится active wiki-карточкой только после проверки, решения владельца и отдельного переноса в нужный слой. Если идея устарела или отклонена, перенести в Архив/К разбору/Отклонено или другой закрытый архив входов.

## Идея

Проверить Box как потенциально более дешевый букс для CPA/заданий GoldenConnect.

## Почему может быть ценно

- В сыром пакете Box отмечен как возможная альтернатива SurfEarner-активностям через менеджера.
- Может быть дешевле для конкретных действий, если поддерживает самостоятельный запуск заданий.

## К чему относится

- GoldenConnect.
- Дешевые источники трафика.

## Что проверить

- доступные форматы заданий;
- минимальный бюджет;
- цена валидного действия;
- возможность UTM;
- прохождение цепочки до Telegram и подтверждения спонсора.

## Как понять, что идея сработала

Есть отдельный тест с понятной ценой, метками и результатом по валидным действиям.

## Риски или сомнения

- Запущенного теста пока нет.
- Качество аудитории может быть низким.
