# Линейка Python торговых ботов

Страница фиксирует инженерную классификацию Python-ботов из ручного разбора ChatGPT.

## Верхнее деление

Python-боты делятся на две группы:

1. Сигнальные ручные боты.
2. Полный автомат.

Это деление важно для документации, тестирования и оценки риска: ручной запуск меняет требования к стратегии.

## Сигнальные ручные боты

Сигнальные боты запускаются пользователем по внешнему сигналу или рыночному событию.

Назначение:

- быстро отработать торговую идею;
- использовать мини-сетку, трейлинг, BE или хеджирование как защиту;
- не заменять качество входного сигнала автоматической логикой.

Упомянутые направления:

- `Signal Fork`;
- `Mini Grid BUY / SELL`;
- `News Impact Bot`.

Связанный текущий раздел:

- [News Bot](../../../News%20Bot.md) - ручная новостная bracket-скобка на Python/Backtrader.

## Полный автомат

Автоматические боты должны работать без постоянного ручного запуска.

Упомянутые направления:

- `Barracuda Grid`;
- `Quantum Hedge`;
- `Chandler Exit / CE Adaptive Strategy`;
- `Bollinger Buster ADX`;
- `Weekend Barracuda`.

Связь с текущей wiki:

- `Barracuda Grid` относится к [TiTan Atlas (BA)](../../../TiTan%20Atlas%20(BA).md).
- `Chandler Exit / CE Adaptive Strategy` относится к [TiTan Kronos (CE)](../../../TiTan%20Kronos%20(CE).md).
- `Bollinger Buster ADX` относится к [TiTan Helios (Bollinger ADX)](../../../TiTan%20Helios%20(Bollinger%20ADX).md).
- `Weekend Barracuda` пока фиксируется как идея криптовалютной сеточной версии, но не как отдельный подтвержденный бот.
- `Quantum Hedge` пока фиксируется как идея хедж-связки, а не как готовый бот.

## Signal Fork

Тип: сигнальный ручной бот / hedge fork.

Идея:

- одновременно запускаются BUY и SELL;
- каждая сторона имеет свою мини-сетку;
- прибыльная сторона должна компенсировать ошибочное направление;
- мини-сетка помогает пережить боковик;
- trailing-BE должен закрывать позицию без сильной просадки после первого удачного движения.

Критичные проверки:

- синхронность запуска;
- корректные параметры BUY и SELL;
- поведение в узком боковике;
- поведение в резком направленном движении;
- комиссии и спред;
- риск двойного стопа в пиле.

## Mini Grid BUY / SELL

Тип: сигнальный ручной бот.

Идея:

- запуск в одну сторону по сигналу;
- мини-сетка помогает усреднить вход;
- при откате бот может выйти лучше, чем одиночный ручной вход;
- trailing-BE защищает после первого профита;
- forced-exit нужен на случай глубокого движения против позиции.

## Quantum Hedge

Тип: хедж-связка на обратнокоррелирующих активах.

Идея:

- запуск на двух инструментах;
- снижение суммарной просадки;
- возможность более агрессивной настройки за счет хеджирования.

Критичные риски:

- корреляция может ломаться;
- оба инструмента могут одновременно уйти против стратегии;
- разные волатильности требуют нормализации лота;
- нужно считать не только profit, но и combined DD.

## Weekend Barracuda

Тип: криптовалютная версия сеточного бота.

Идея:

- торговать криптовалюту в выходные;
- использовать периоды, когда Forex закрыт;
- собирать микро-движения и боковики.

Критичные риски:

- комиссии на криптобирже;
- проскальзывание;
- резкие новости по BTC;
- ночная ликвидность;
- ложное ощущение низкой волатильности.
