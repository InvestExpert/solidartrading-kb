# ATLAS Crypto - архитектура спотовой BUY-сетки

## Суть

ATLAS SPOT BUY - это адаптация BUY-only сетки [TiTan Atlas (BA)](../../../TiTan%20Atlas%20(BA).md) к крипто-споту.

Архитектурная идея:

- работать только в покупку;
- торговать Spot без плеча;
- не иметь риска margin call;
- набирать позицию по сетке;
- фиксировать прибыль на откатах;
- в глубокой просадке переходить в холд монеты.

## Почему Spot подходит ATLAS BUY

ATLAS работает только в покупку. На Forex BUY-сетка использует маржинальную модель: если просадка становится слишком глубокой, счет может упереться в маржу.

На Spot покупка означает владение активом. Поэтому крайний сценарий другой:

```text
не принудительное закрытие и смерть депозита, а удержание монеты.
```

Это делает Spot логичной средой для BUY-only сетки.

## Базовые элементы

- Только BUY-направление.
- Spot без плеча.
- Ликвидные топовые монеты.
- Портфель из нескольких активов.
- Лимит капитала на один актив.
- Контроль средней цены входа.
- Контроль доли капитала в монете.
- Контроль времени нахождения в холде.
- Фиксация прибыли на откатах.
- Сравнение результата с buy-and-hold.

## Отличие от ходлинга

Ходлинг:

- покупает актив;
- ждет роста;
- прибыль появляется только при росте выше цены покупки.

ATLAS SPOT BUY:

- набирает позицию по сетке;
- снижает среднюю цену входа;
- продает часть позиции на откатах;
- может извлекать прибыль из волатильности до полного восстановления актива.

Короткая формула:

```text
ATLAS на споте - не просто купить и ждать. В крайнем сценарии он становится холдом, но с лучшей средней ценой входа.
```

## Метрики проверки

Для проверки архитектуры нужны:

- итоговая equity;
- максимальная просадка;
- время восстановления;
- средняя цена входа;
- доля капитала в монете;
- доля капитала в USDT;
- количество прибыльных SELL;
- срок нахождения капитала в холде;
- сравнение с buy-and-hold;
- портфельная просадка по нескольким активам.

## Ограничения

- Не использовать случайные слабые альты.
- Не держать весь капитал в одной монете.
- Не переносить параметры с Forex без тестов.
- Не смешивать Spot и Futures как одинаковые рынки.
- Не считать отсутствие ликвидации гарантией роста актива.
