# 2. Обмен трафиком / серфинг-сети / автосерфинг

## Суть

Групповая карточка первичного реестра источников трафика. Это список для изучения и будущих тестов, а не подтверждение эффективности.

## Макро-группа

[01 Мотивированный трафик и задания](./01%20%D0%9C%D0%BE%D1%82%D0%B8%D0%B2%D0%B8%D1%80%D0%BE%D0%B2%D0%B0%D0%BD%D0%BD%D1%8B%D0%B9%20%D1%82%D1%80%D0%B0%D1%84%D0%B8%D0%BA%20%D0%B8%20%D0%B7%D0%B0%D0%B4%D0%B0%D0%BD%D0%B8%D1%8F%20-%20%D0%98%D0%BD%D0%B4%D0%B5%D0%BA%D1%81.md)

## Первичная оценка

| Поле | Значение |
|---|---|
| Скорость запуска | быстро |
| Примерное качество | очень низкое |
| Уровень конкуренции | низкая |
| Риск мусорного трафика | очень высокий |
| Подходит для теста бесплатной регистрации | только как контроль |
| Подходит для платного оффера | нет |
| Нужен tracker / redirect / UTM | да |
| Статус | осторожно / не использовать |

## Пометка из исходного промта

скорее контрольный мусорный трафик, не смешивать с нормальными источниками.

## Что проверить

- совпадение аудитории с проектом;
- качество действия после первого перехода;
- риск фрода, накрутки или мотивированного мусора;
- возможность раздельного tracking / UTM;
- применимость к GoldenConnect, SolidarTrading и универсальным тестам отдельно.

## GoldenConnect - активный тест SurfEarner

SurfEarner / ADV24 используется как активный тестовый источник для GoldenConnect в формате [живых просмотров сайта](./SurfEarner%20-%20%D0%B6%D0%B8%D0%B2%D1%8B%D0%B5%20%D0%BF%D1%80%D0%BE%D1%81%D0%BC%D0%BE%D1%82%D1%80%D1%8B%20%D1%81%D0%B0%D0%B9%D1%82%D0%B0.md).

Проверяются три длительности просмотра:

- [15 секунд](../../%D0%A2%D0%B5%D1%81%D1%82%D1%8B/T-GC-SURF-001%20-%20SurfEarner%20-%20%D0%B6%D0%B8%D0%B2%D1%8B%D0%B5%20%D0%BF%D1%80%D0%BE%D1%81%D0%BC%D0%BE%D1%82%D1%80%D1%8B%2015%20%D1%81%D0%B5%D0%BA%D1%83%D0%BD%D0%B4.md);
- [30 секунд](../../%D0%A2%D0%B5%D1%81%D1%82%D1%8B/T-GC-SURF-002%20-%20SurfEarner%20-%20%D0%B6%D0%B8%D0%B2%D1%8B%D0%B5%20%D0%BF%D1%80%D0%BE%D1%81%D0%BC%D0%BE%D1%82%D1%80%D1%8B%2030%20%D1%81%D0%B5%D0%BA%D1%83%D0%BD%D0%B4.md);
- [60 секунд](../../%D0%A2%D0%B5%D1%81%D1%82%D1%8B/T-GC-SURF-003%20-%20SurfEarner%20-%20%D0%B6%D0%B8%D0%B2%D1%8B%D0%B5%20%D0%BF%D1%80%D0%BE%D1%81%D0%BC%D0%BE%D1%82%D1%80%D1%8B%2060%20%D1%81%D0%B5%D0%BA%D1%83%D0%BD%D0%B4.md).

Граница: не считать визит результатом. Сравнивать длительности по переходам в Telegram и дальнейшим действиям.

## Граница

Не считать источники из этой карточки рабочими до отдельного теста или решения владельца. Не создавать рекламные тексты, офферы и бизнес-обещания на основании этой карточки.
