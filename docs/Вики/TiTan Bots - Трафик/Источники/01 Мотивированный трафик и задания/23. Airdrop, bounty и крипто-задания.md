| Поле | Значение |
|---|---|
| Скорость запуска | быстро |
| Примерное качество | мотивированное / crypto |
| Уровень конкуренции | средняя |
| Риск мусорного трафика | высокий |
| Подходит для теста бесплатной регистрации | да |
| Подходит для платного оффера | нет / осторожно |
| Нужен tracker / redirect / UTM | да |
| Статус | тестировать осторожно |

## Пометка из исходного промта

если проект можно упаковать под Web3/bonus/referral механику.

## Что проверить

- совпадение аудитории с проектом;
- качество действия после первого перехода;
- риск фрода, накрутки или мотивированного мусора;
- возможность раздельного tracking / UTM;
- применимость к GoldenConnect, SolidarTrading и универсальным тестам отдельно.
