## Суть формата

Стандартное задание в SocPublic позволяет оплатить конкретное действие исполнителя и проверить результат вручную. Для GoldenConnect это лучше, чем показы, письма или баннеры, потому что первый тест должен проверить прохождение цепочки, а не просто клики.

## Рабочая цепочка SOC-01

```text
SocPublic
-> стандартное задание
-> страница захвата GoldenConnect
-> Telegram-бот
-> канал / чат
-> подтверждение спонсора
-> скрин + Telegram-ник
```

## Тип задания

Выбранный тип в форме:

```text
Прочая комбинация типов
```

Почему:

- есть переход на сайт;
- есть действие на сайте;
- есть переход в Telegram;
- есть запуск Telegram-бота;
- есть вступление в канал;
- есть подключение к чату;
- есть подтверждение спонсора;
- есть отчет со скрином.

Не выбирать как простую регистрацию без активности: цепочка требует действий в Telegram.

## Название задания

Рабочее название:

```text
Сайт, Telegram-бот и подписка
```

## Цена

Наблюдаемые ориентиры по заданиям внутри SocPublic:

| Тип задания | Ориентир цены |
|---|---:|
| Простая регистрация без активности | 2-3.5 руб. |
| Telegram-бот / переход в бота | около 4 руб. |
| Регистрация + подтверждение / скрин / активность | 5-10 руб. |
| Сложные регистрации / документы / финансы | 20-30 руб. и выше |

Рабочее решение для SOC-01:

- 7 руб. исполнителю;
- 9.10 руб. стоимость для рекламодателя;
- 20 выполнений как полный первый лимит;
- начать можно со 100 руб. и проверить первые отчеты;
- если берут медленно, рассмотреть 10 руб.;
- если много мусора, сначала править инструкцию и проверку, а не цену.

## Текст задания

Использовать без картинок на первом запуске.

```text
1. Перейдите по ссылке из задания.

2. На открывшейся странице нажмите кнопку «Показать как» или прокрутите страницу немного вниз, чтобы появилось верхнее меню.

3. В верхнем меню нажмите кнопку «Телеграм».

4. Откроется Telegram-бот. Нажмите Start / Старт.

5. После запуска бот попросит вступить в канал и чат.

6. Нажмите «Канал» и в Telegram-канале нажмите «Присоединиться».

7. Вернитесь в Telegram-бот.

8. Нажмите «Чат». Вас перекинет в чат вашего языка.

9. В чате слева выберите любую тему и нажмите «Подключиться».

10. Вернитесь в Telegram-бот и нажмите «Я вступил(а)».

11. Бот покажет сообщение «Подтверждение спонсора».

12. Проверьте, что указан спонсор: Alexey Titov / @InvestExpert_info.

13. Нажмите «Подтвердить».

14. После подтверждения сделайте один скриншот, где видно сообщение со спонсором и сообщение, что спонсор подтверждён.

15. Прикрепите этот скриншот в отчёт и напишите свой ник в Telegram.

Важно:
- ничего покупать не нужно;
- банковские данные вводить не нужно;
- установка программ не требуется;
- задание оплачивается только при правильном скриншоте и Telegram-нике.
```

## Подтверждение задания

```text
Для подтверждения задания пришлите:

1. Один скриншот из Telegram-бота, где одновременно видно:
- сообщение «Подтверждение спонсора»;
- спонсора Alexey Titov / @InvestExpert_info;
- сообщение после подтверждения, что спонсор подтверждён.

2. Свой ник в Telegram, например:
@username

Задание не оплачивается, если:
- нет ника в Telegram;
- нет скриншота;
- на скриншоте не видно спонсора Alexey Titov / @InvestExpert_info;
- не видно, что спонсор подтверждён;
- скриншот обрезан или нечитаемый;
- прислан чужой или повторяющийся скриншот.
```

## Решения после первых отчетов

| Результат | Действие |
|---|---|
| быстро выполняют, скрины норм | продолжить до 20 |
| мало берут | поднять оплату до 10 руб. |
| много мусора | уточнить описание / подтверждение |
| нет Telegram-ника | отправлять на доработку |
| скрин без спонсора | доработка или отказ |
| скрин нечитаемый | доработка |
| есть скрин, но нет в базе | проверить ссылку / бот / спонсора |
| люди не понимают чат / темы | добавить пояснение в описание |
| модерация отклоняет | смягчить формулировки и подать как одну цепочку Telegram-бота |
