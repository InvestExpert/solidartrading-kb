# 25. Офлайн / локальные / нестандартные источники

## Суть

Групповая карточка первичного реестра источников трафика. Это список для изучения и будущих тестов, а не подтверждение эффективности.

## Первичная оценка

| Поле | Значение |
|---|---|
| Скорость запуска | медленно |
| Примерное качество | локальное / точечное |
| Уровень конкуренции | низкая |
| Риск мусорного трафика | низкий / средний |
| Подходит для теста бесплатной регистрации | да |
| Подходит для платного оффера | осторожно |
| Нужен tracker / redirect / UTM | желательно |
| Статус | резерв |

## Дополнено из видео

Источник: [ИСТОЧНИКИ трафика для АРБИТРАЖА Трафика](https://www.youtube.com/watch?v=UyljNNG49iQ), обработано 2026-06-26.

- QR-коды на авто.
- Сайты знакомств как нестандартный ручной источник. Не рекомендация: использовать только как риск-гипотезу и отдельно проверять правила площадки, согласие пользователя и репутационные риски.
- AirDrop / прямое распространение ссылок как near-field источник. Не рекомендация: высокий риск спама и слабая масштабируемость.

Источник: [12 бесплатных источников трафика для бизнеса в 2026](https://www.youtube.com/watch?v=gz1MtJYXcNg), обработано 2026-06-26.

- Яндекс Бизнес, Яндекс Карты, 2ГИС и городские площадки учитывать как локальный traffic-layer для услуг, офлайн-точек и региональных предложений.
- Локальные сообщества, локальные сайты и локальные форумы могут давать точечный спрос, но не масштабируются как рекламный кабинет.
- Для локальных источников важны не только клики, но и звонки, маршруты, сообщения, отзывы, профильные просмотры и качество заявки.
- Для онлайн-проектов вроде GoldenConnect или SolidarTrading локальные источники не считать приоритетом без отдельной проектной гипотезы.

Источник: [ВСЕ ИСТОЧНИКИ ТРАФИКА: ТИР ЛИСТ](https://www.youtube.com/watch?v=UhrO634QexU), обработано 2026-06-26.

- Офлайн-трафик и ближний круг учитывать как отдельный ручной источник для первых действий, простых CPA/referral задач и локальных проверок.
- Подтипы: друзья и знакомые, мероприятия, бизнес-завтраки, коворкинги, локальные предпринимательские группы, офлайн-мотивированные действия.
- Ограничения: небольшой объем, зависимость от личного доверия, репутационный ущерб при слабом продукте или агрессивной подаче.
- Нетворкинг учитывать как отношенческий источник клиентов, партнеров и коллабораций, а не как массовый источник кликов.

## Форматы трафика

- Форматы в исходном промте не выделены отдельным списком.

## Пометка из исходного промта

не основной онлайн-трафик, но занести как нестандартный слой.

## Что проверить

- совпадение аудитории с проектом;
- качество действия после первого перехода;
- риск фрода, накрутки или мотивированного мусора;
- возможность раздельного tracking / UTM;
- применимость к GoldenConnect, SolidarTrading и универсальным тестам отдельно.

## Граница

Не считать источники из этой карточки рабочими до отдельного теста или решения владельца. Не создавать рекламные тексты, офферы и бизнес-обещания на основании этой карточки.
