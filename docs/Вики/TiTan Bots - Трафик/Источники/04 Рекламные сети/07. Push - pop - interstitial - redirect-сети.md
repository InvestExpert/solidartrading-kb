# 7. Push / pop / interstitial / redirect-сети

## Суть

Групповая карточка первичного реестра источников трафика. Это список для изучения и будущих тестов, а не подтверждение эффективности.

## Первичная оценка

| Поле | Значение |
|---|---|
| Скорость запуска | быстро |
| Примерное качество | низкое / смешанное |
| Уровень конкуренции | средняя |
| Риск мусорного трафика | высокий |
| Подходит для теста бесплатной регистрации | да |
| Подходит для платного оффера | нет / осторожно |
| Нужен tracker / redirect / UTM | да |
| Статус | осторожно |

## Пометка из исходного промта

низкая цена, много мусора, обязательно отдельные тесты и blacklist площадок.

## Что проверить

- совпадение аудитории с проектом;
- качество действия после первого перехода;
- риск фрода, накрутки или мотивированного мусора;
- возможность раздельного tracking / UTM;
- применимость к GoldenConnect, SolidarTrading и универсальным тестам отдельно.

## Граница

Не считать источники из этой карточки рабочими до отдельного теста или решения владельца. Не создавать рекламные тексты, офферы и бизнес-обещания на основании этой карточки.
