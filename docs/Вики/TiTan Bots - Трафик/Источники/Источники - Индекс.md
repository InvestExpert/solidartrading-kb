# Источники - Индекс

Закрытый подраздел для профилей источников и групп источников трафика.

## Правило раздела

Группы источников заведены как верхняя классификация для будущего наполнения. Внутри группы пока хранится только описание логики, аудитории, offer-fit, project-fit, методов и рисков.

Конкретный источник получает отдельную карточку только когда по нему появляется собственный материал: условия, контакты, тест, бюджет, результат, причина отклонения или повторяемое решение.

Карточка источника хранит гипотезу применимости, а не результат эффективности.
