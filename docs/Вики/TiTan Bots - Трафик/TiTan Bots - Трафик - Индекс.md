# TiTan Bots - Трафик - Индекс

Закрытый верхний слой wiki TiTan Bots для источников трафика, гипотез применимости, тестов рекламы, связок, метрик и результатов.

Раздел не является публичным слоем и не должен становиться FAQ/RAG-маршрутом для внешнего пользователя. Публичные страницы, офферы и объяснения остаются в [TiTan Bots - Метод и продвижение](../TiTan%20Bots%20-%20%D0%9C%D0%B5%D1%82%D0%BE%D0%B4%20%D0%B8%20%D0%BF%D1%80%D0%BE%D0%B4%D0%B2%D0%B8%D0%B6%D0%B5%D0%BD%D0%B8%D0%B5/TiTan%20Bots%20-%20%D0%9C%D0%B5%D1%82%D0%BE%D0%B4%20%D0%B8%20%D0%BF%D1%80%D0%BE%D0%B4%D0%B2%D0%B8%D0%B6%D0%B5%D0%BD%D0%B8%D0%B5%20-%20%D0%98%D0%BD%D0%B4%D0%B5%D0%BA%D1%81.md).

## Что сюда класть

- профили источников трафика;
- группы источников и предварительные гипотезы по аудитории;
- внутренние traffic/CPA/advertising тесты;
- связки источник -> аудитория -> входной оффер -> посадочная точка;
- метрики качества, результаты и причины отклонения;
- проектные ветки GoldenConnect, SolidarTrading и универсальных тестов.

## Граница с тестами

Карточка источника хранит предварительную гипотезу применимости: какая аудитория там вероятна, какой входной оффер может подойти и для каких проектов источник выглядит перспективным.

Результат эффективности хранится только в тесте или реестре тестов после фактической проверки.

## Граница с оценками и связками

Оценки, выводы и связки пока не выделяются в отдельную сложную структуру. Их модель будет выбрана после первых рабочих гипотез, комбинаций и результатов, чтобы не закрепить преждевременную схему.
