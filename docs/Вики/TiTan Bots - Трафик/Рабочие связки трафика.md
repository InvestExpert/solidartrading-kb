# Рабочие связки трафика

Реестр связок, которые дали полезный результат или имеют подтвержденную причину для повторной проверки.

Детальная модель связок пока не закреплена. Вернуться к структуре после первых гипотез, тестов и повторяемых комбинаций.

## Формат связки

- источник;
- аудитория;
- входной оффер;
- посадочная точка;
- проект;
- метрики результата;
- ограничения;
- что можно повторить;
- что нельзя переносить на другие проекты без теста.
