# GoldenConnect - SurfEarner - тестовый пакет живых просмотров

## Суть

Тестовый пакет GoldenConnect через [SurfEarner / ADV24](../../%D0%98%D1%81%D1%82%D0%BE%D1%87%D0%BD%D0%B8%D0%BA%D0%B8/01%20%D0%9C%D0%BE%D1%82%D0%B8%D0%B2%D0%B8%D1%80%D0%BE%D0%B2%D0%B0%D0%BD%D0%BD%D1%8B%D0%B9%20%D1%82%D1%80%D0%B0%D1%84%D0%B8%D0%BA%20%D0%B8%20%D0%B7%D0%B0%D0%B4%D0%B0%D0%BD%D0%B8%D1%8F/SurfEarner%20-%20%D0%B8%D1%81%D1%82%D0%BE%D1%87%D0%BD%D0%B8%D0%BA%20%D1%82%D1%80%D0%B0%D1%84%D0%B8%D0%BA%D0%B0.md). Это не рабочая связка и не подтверждение эффективности. Пакет нужен, чтобы сравнить три длительности живого просмотра страницы захвата: 15, 30 и 60 секунд.

## Цепочка

```text
SurfEarner / ADV24
-> живой просмотр страницы
-> страница захвата GoldenConnect
-> кнопка / переход в Telegram
-> Telegram-бот
-> канал / чат
-> подтверждение спонсора
```

## Что проверяем

- дает ли дешевый просмотр страницы переход в Telegram;
- влияет ли длительность просмотра на Telegram CTR и дальнейшие действия;
- не является ли результат только накруткой визитов и времени на странице;
- нужно ли дальше тестировать SurfEarner без гео или с другими настройками.
