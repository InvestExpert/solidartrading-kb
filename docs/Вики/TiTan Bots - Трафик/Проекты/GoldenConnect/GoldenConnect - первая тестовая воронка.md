# GoldenConnect - первая тестовая воронка

## Суть

Рабочая карточка первой тестовой воронки GoldenConnect для дешевого трафика. Это закрытая traffic-операционка: она не меняет публичную страницу захвата, оффер, цены, обещания и бизнес-смысл.

## Базовая цепочка

```text
источник трафика
-> страница захвата GoldenConnect
-> переход в Telegram
-> запуск Telegram-бота
-> прогрев сообщениями
-> реферальное действие / удержание / повторное касание
```

## Почему GoldenConnect подходит для первого дешевого теста

- вход формально бесплатный или низкопороговый;
- целевая аудитория шире, чем у дорогого инвестиционного оффера;
- Telegram-бот может догревать слабый или холодный вход;
- можно проверять не только покупательскую готовность, но и механику рефералов;
- дешевые источники позволяют быстро проверить карточки рекламы, страницу захвата и переход в Telegram.

## Что проверяет первая воронка

Первая цель - не доказать прибыльность источника, а отладить измеримый процесс:

- какой источник дает клики;
- какой формат рекламы пропускает модерация;
- какой текст дает переход на страницу;
- какой процент доходит до Telegram;
- какой процент запускает бота;
- остается ли пользователь после первого дня;
- есть ли реферальное действие;
- какие данные нужно фиксировать для повторения теста.

## Основные метрики

| Этап | Что считать |
|---|---|
| Источник | показы, клики, расход, формат, модерация |
| Страница захвата | переходы, отказы, клики в Telegram |
| Telegram | старты бота, подписки, блокировки, удержание 1/3/7 дней |
| Прогрев | просмотры сообщений, реакции, переходы дальше |
| Реферальная механика | люди под пользователем, первое реферальное действие, повторные действия |
| Качество | дубли, мусор, жалобы, фрод, ручная оценка диалогов |

## Стартовая логика

Первый тест делать на одной странице захвата и одном Telegram-route. Менять одновременно только один крупный элемент:

- источник;
- рекламный угол;
- формат объявления;
- текст;
- посадочная точка.

Если одновременно менять источник, текст, страницу и бота, результат нельзя будет сравнить.

## Что не считать успехом

- много кликов без перехода в Telegram;
- много стартов бота без удержания;
- дешевые регистрации без реферального действия;
- рост цифр, который держится только на оплате задания;
- источник, где невозможно понять формат, расход и метку.
