# Реестр источников трафика

Рабочий реестр источников и групп источников. Реестр не заполняется большим списком без отдельного подтверждения.

## Поля карточки источника

- название источника или группы;
- тип источника;
- статус: candidate / active_test / tested / rejected / paused;
- вероятная аудитория;
- предварительно лучше подходит для: GoldenConnect / SolidarTrading / универсальные тесты / другое;
- вероятный входной оффер;
- гипотеза применимости;
- риски качества;
- что проверить тестом;
- связанные тесты;
- итог после тестов.

## Источники

Конкретные площадки из первичного промта занесены в групповые карточки 25 групп ниже. Отдельные карточки по каждой площадке не создавались.
