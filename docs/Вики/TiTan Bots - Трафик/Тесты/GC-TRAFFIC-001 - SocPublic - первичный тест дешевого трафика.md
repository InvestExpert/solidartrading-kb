# GC-TRAFFIC-001 - SocPublic - первичный тест дешевого трафика

## Статус

active_test / задание создано в SocPublic; UTM виден в Метрике, полных результатов качества пока нет.

## Суть

Первый тест дешевого трафика GoldenConnect через SocPublic / Соцпаблик. Это карточка запуска и результата по SOC-01. Цены, формат, UTM и инструкция внесены по сырому ChatGPT-разбору от 2026-07-02; промежуточный сигнал из Метрики внесен по пакету `traffic_goldenconnect_codex_import_2026-07-02.md`.

## Гипотеза

SocPublic как букс / площадка заданий может дешево привести пользователей на страницу захвата GoldenConnect и помочь проверить:

- рекламную карточку;
- переход на страницу;
- переход в Telegram;
- запуск бота;
- удержание после первого касания;
- реферальную механику;
- сам процесс фиксации теста.

Ожидаемое качество аудитории: низкое или смешанное. Главная ценность теста - отладка воронки и учета.

## Цепочка теста

```text
SocPublic
-> стандартное задание
-> страница захвата GoldenConnect
-> Telegram
-> бот
-> канал / чат
-> подтверждение спонсора
-> скрин + Telegram-ник
```

## Правила проверки отчета

Для принятия нужен один скриншот из Telegram-бота, где одновременно видно:

- сообщение `Подтверждение спонсора`;
- спонсора `Alexey Titov / @InvestExpert_info`;
- сообщение после подтверждения, что спонсор подтвержден.

Также исполнитель должен отдельно написать Telegram-ник.

Не оплачивать без конкретного комментария нельзя. Если нет ника, нет скрина, не виден спонсор, не видно подтверждение, скрин обрезан/нечитаемый или повторяется, отправлять на доработку или отклонять по факту дефекта.
