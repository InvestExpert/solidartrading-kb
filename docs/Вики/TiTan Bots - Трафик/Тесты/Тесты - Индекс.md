# Тесты - Индекс

Закрытый подраздел для фактических traffic/CPA/advertising тестов.

## Статус структуры

Детальная структура тестов пока не фиксируется. Ее нужно проектировать отдельно после первых рабочих цепочек `проект -> оффер -> источник трафика -> метод -> результат`.
