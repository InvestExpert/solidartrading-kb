# Связки - Индекс

Закрытый подраздел для связок источник -> аудитория -> оффер -> посадочная точка.

## Статус структуры

Детальная структура связок пока не фиксируется. Связки могут оказаться частью тестовой модели или отдельным слоем после первых рабочих комбинаций.

Схемы из внешних материалов сохраняются отдельно от рабочих связок. Они нужны, чтобы не терять готовые воронки вида `оффер -> источник -> посадочная точка -> заявленный результат`, но не являются нашими результатами до теста.
