# Отклоненные источники трафика

Реестр источников, групп источников и подходов, которые не стоит повторять без новой причины.

## Формат записи

- источник или группа;
- проект;
- что тестировали;
- почему отклонено;
- какие условия могут вернуть источник в работу;
- ссылка на тест или решение.

## Источники

Пока нет отклонений по результатам наших тестов.

## Предварительные стоп-подходы

Эти пункты не являются результатом теста. Это подходы, которые нельзя случайно принять за рабочий источник без отдельного решения владельца и legal-check.

| Подход | Причина |
|---|---|
| Покупка нелегальных баз | Юридический и репутационный риск |
| SMS без opt-in | Спам, жалобы, блокировки и юридический риск |
| Холодный обзвон без согласия | Юридический и репутационный риск |
| Fingerprint-like идентификация без правового основания | Privacy-риск |
| Прямой финансовый оффер на неизвестную аудиторию | Высокий риск промаха, слабого CR и плохого качества |
| Обход банов, клоакинг, фейковые аккаунты и серые расходники | Нарушение правил площадок, блокировки, репутационный риск |
| Копирование чужих креативов без права использования | Юридический и репутационный риск, слабая переносимость результата |
| Перезалив чужого контента | Авторские права, баны, слабая переносимость результата |
| Использование чужих лиц или фейковых экспертов в AI UGC | Юридический и репутационный риск, ввод пользователя в заблуждение |
| Telegram-бот с логикой давления или продажи любой ценой | Репутационный риск и риск жалоб |
| Перенос бюджета на видеоплощадку только по числу просмотров | Просмотры без вовлеченности и переходов не доказывают качество трафика |
| Взаимные подписки в Telegram | Мотивированный мусорный трафик, портит качество аудитории и статистику канала |
| Ручное добавление людей в Telegram-канал без согласия | Жалобы, блокировки, репутационный риск |
| Спам по Telegram-чатам | Spam block, жалобы админов, низкое качество аудитории |
| Автоспам-боты в Telegram-комментариях | Спам, жалобы, блокировки, скам-ассоциация и низкое качество аудитории |
| Парсинг участников групп с последующей рассылкой | Нелегальная или спорная база, спам, жалобы и репутационный риск |
| Фейковые или серые массовые рассылки | Нет подтвержденного согласия, высокий риск блокировок и жалоб |
