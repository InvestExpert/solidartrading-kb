# Telegram-стикеры с AI-персонажем SolidarTrading

## Что это

Пакет Telegram-стикеров с единым AI-персонажем SolidarTrading.

Пакет используется как готовый промо-материал для Telegram-коммуникации: реакции, напоминания, приветствия, рабочие сообщения и эмоциональные вставки в чатах.

Стикеры созданы по внутреннему процессу создания Telegram-стикеров с AI-персонажем. В этой карточке процесс не линкуется, чтобы промо-материалы не тянули служебный слой `Система` в публичный граф.

## Состав пакета

Файлы лежат рядом с карточкой:

```text
Telegram-стикеры с AI-персонажем - файлы/
├── Базовый персонаж/
├── Статические PNG/
└── Анимированные WEBM/
```

На момент импорта:

- базовые референсы персонажа: `10` файлов;
- статические PNG-стикеры: `37` файлов;
- готовые анимированные WEBM-стикеры: `12` файлов.

## Базовый персонаж

Папка `Базовый персонаж` хранит референсы, которые нужны для повторной генерации стикеров в одном стиле.

Референсы не являются отдельными клиентскими материалами. Их задача - удерживать единый образ персонажа: лицо, очки, броня, цветовые акценты и общий comic cartoon style.

## Где использовать

- Telegram-чат проекта;
- Telegram-канал;
- прогревочные сообщения;
- напоминания перед созвоном;
- реакции в команде и с пользователями;
- визуальная поддержка AI-персонажа SolidarTrading.

## Проверено при импорте

- Пакет найден в пользовательской папке стикеров.
- Медиафайлы отделены от `.md` raw-файлов, которые лежали в той же папке.
- WEBM-файлы проверены по размеру: каждый меньше `256 KB`.
- Файлы перенесены из сырья в `Материалы/Стикеры`.

## Не проверено

- Кодек WEBM, alpha-канал, FPS, длительность и аудиопоток локально не проверены, потому что `ffprobe` не найден в текущем окружении.
- Telegram pack URL не указан.
