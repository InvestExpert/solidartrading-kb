# SolidarTrading - шаблон пульса счета для Telegram

## RAG-контекст

Эта карточка - risk-boundary по теме "SolidarTrading - шаблон пульса счета для Telegram". RAG должен сохранять ограничения и не превращать рискованные формулировки в доказанное обещание.

## Назначение

Это рабочий шаблон публичного Telegram-статуса по состоянию счета.

Он не является персональной финансовой рекомендацией. Пост сообщает режим счета и действия команды, но не командует участникам пополнять или выводить деньги.

Каноническое правило: [../../Метод/SolidarTrading - управление рабочим депозитом резервом и режимами просадки](../../%D0%9C%D0%B5%D1%82%D0%BE%D0%B4/SolidarTrading%20-%20%D1%83%D0%BF%D1%80%D0%B0%D0%B2%D0%BB%D0%B5%D0%BD%D0%B8%D0%B5%20%D1%80%D0%B0%D0%B1%D0%BE%D1%87%D0%B8%D0%BC%20%D0%B4%D0%B5%D0%BF%D0%BE%D0%B7%D0%B8%D1%82%D0%BE%D0%BC%20%D1%80%D0%B5%D0%B7%D0%B5%D1%80%D0%B2%D0%BE%D0%BC%20%D0%B8%20%D1%80%D0%B5%D0%B6%D0%B8%D0%BC%D0%B0%D0%BC%D0%B8%20%D0%BF%D1%80%D0%BE%D1%81%D0%B0%D0%B4%D0%BA%D0%B8.md).

## Цвета режима

### Зеленый

```text
Пульс счета SolidarTrading: зеленый

Счет находится в штатной зоне. Команда продолжает обычный мониторинг. Никаких специальных действий от участников не требуется.

Можно ориентироваться на текущий режим, но любые решения по своим средствам участник принимает самостоятельно.
```

### Желтый

```text
Пульс счета SolidarTrading: желтый

Счет подошел к зоне повышенного внимания. Команда чаще проверяет Balance, Equity, открытый минус, позиции и Free Margin. Резерв готовится как защитный инструмент, но это еще не команда участникам что-либо доливать.

Можно ориентироваться на режим и не делать хаотичных выводов или пополнений без понимания состояния счета.
```

### Оранжевый

```text
Пульс счета SolidarTrading: оранжевый

Счет находится в зоне, где команда может принимать решения по временному усилению master-счета. Режим определяется по фактическим данным RoboForex, а не по Myfxbook-процентам.

Для участников это ориентир по состоянию системы, а не индивидуальная команда пополнить счет.
```

### Красный

```text
Пульс счета SolidarTrading: красный

Счет находится в высокой зоне риска и требует частого контроля. Команда оценивает открытые позиции, свободную маржу, скорость изменения просадки и необходимость защитных действий по master-счету.

Участникам не стоит принимать резкие решения по выводу или доливу только по эмоциям. Это не персональная рекомендация; при сомнениях лучше задать вопрос в общем чате.
```

### Ручной

```text
Пульс счета SolidarTrading: ручной контроль

Счет находится в зоне, где решения принимаются только вручную. Механические правила и обещания здесь недопустимы: команда оценивает фактическое состояние счета, позиции, маржу и рыночную ситуацию.

Публичный статус сообщает состояние системы. Он не является командой участникам пополнять или выводить средства.
```

## Запрещенные формулировки

Не писать:

- `всем долить`;
- `всем вывести`;
- `счет в безопасности`;
- `просадка точно восстановится`;
- `долив решит проблему`;
- `это правильное решение для каждого`.

Использовать:

- `можно ориентироваться`;
- `решение принимает участник`;
- `это не индивидуальная рекомендация`;
- `команда мониторит`;
- `команда принимает решение по master-счету`.
