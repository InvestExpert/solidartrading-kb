## Главный ответ

SolidarTrading подходит нескольким типам людей, но каждому нужен свой маршрут.

Основной публичный формат для обычного пользователя - подключение TiTan ATLAS через Copy Trading на личном брокерском счете. Покупать робота, устанавливать советника и самостоятельно торговать не нужно.

Дополнительно доступны партнерское направление, роль лидера, отдельная покупка TiTan ATLAS и White Label для команд или компаний. Эти форматы нельзя смешивать в одном первом ответе новичку.

## Основные форматы участия

| Формат | Для кого | Главный смысл | Публичный статус |
| --- | --- | --- | --- |
| Copy Trading | новичок, инвестор, обычный пользователь | робот работает централизованно, сделки копируются на личный счет | основной публичный формат |
| партнерская программа | партнер, блогер, владелец аудитории, человек без собственного депозита | получать вознаграждение от торгового оборота структуры | доступно сейчас |
| лидер команды | партнер, который строит и сопровождает команду | масштабировать партнерское направление и обсуждать индивидуальные условия | доступно через партнерскую модель |
| отдельная покупка TiTan ATLAS | опытный пользователь, трейдер, команда, клиент другого брокера | использовать защищенную версию робота отдельно от публичного Copy Trading | обсуждается индивидуально |
| White Label | лидер, команда, компания, проект | запуск отдельного продукта под своим брендом на базе технологии TiTan Bots | доступно по индивидуальному договору |

## Обычный пользователь без опыта трейдинга

Подходит человеку, который:

- хочет использовать торговую автоматизацию;
- хочет работать через личный брокерский счет;
- не хочет самостоятельно торговать;
- не хочет устанавливать робота и держать включенным компьютер;
- понимает, что рыночный риск остается;
- готов соблюдать правила подключения и управления рабочим балансом.

Короткая формулировка:

```text
Для обычного подключения опыт трейдинга не нужен: робот работает централизованно, а сделки копируются на личный счет автоматически.
```

## Партнер без собственного депозита

Для участия в партнерской программе собственный торговый депозит не обязателен.

Партнер может:

- зарегистрироваться;
- активировать партнерский кабинет;
- получить персональную ссылку;
- приглашать пользователей;
- строить структуру;
- получать партнерское вознаграждение от торгового оборота.

Подробный маршрут хранится в [07_Партнерская программа](../FAQ/07_%D0%9F%D0%B0%D1%80%D1%82%D0%BD%D0%B5%D1%80%D1%81%D0%BA%D0%B0%D1%8F%20%D0%BF%D1%80%D0%BE%D0%B3%D1%80%D0%B0%D0%BC%D0%BC%D0%B0.md).

Короткая формулировка:

```text
Собственный депозит не обязателен для партнерского направления. Если вы хотите использовать робота на своем счете, тогда нужен рабочий депозит.
```

## Лидер команды

Лидер - это партнер, который не только приглашает пользователей, но и строит, сопровождает и развивает команду.

Отдельного формального статуса лидера по умолчанию нет. При этом крупные лидеры могут:

- получить индивидуальные условия;
- обсудить отдельный формат сотрудничества;
- стать частью проекта по индивидуальной договоренности;
- обсудить White Label.

Короткая формулировка:

```text
Формально лидер - это партнер. Отличие в масштабе: лидер строит, сопровождает и развивает команду.
```

## Человек с аудиторией

SolidarTrading может быть интересен:

- блогеру;
- владельцу Telegram-канала;
- лидеру сообщества;
- организатору команды;
- человеку с партнерской структурой;
- специалисту по продвижению.

Главный маршрут:

```text
партнерская программа -> рост команды -> индивидуальные условия -> при необходимости White Label
```

## Компания или проект

Компания может обсуждать White Label, если хочет:

- запустить продукт под своим брендом;
- получить отдельную упаковку;
- использовать технологию TiTan Bots;
- создать отдельную Copy Trading-стратегию;
- подключить собственную аудиторию;
- построить собственную партнерскую структуру.

Короткая формулировка:

```text
White Label - это отдельный продукт под брендом партнера на базе технологии TiTan Bots. Конкретный состав определяется договором.
```

## Опытный пользователь, который хочет купить TiTan ATLAS отдельно

Отдельная покупка подходит человеку или команде, которые:

- хотят самостоятельную работу;
- хотят использовать другого брокера;
- понимают ответственность за настройки и риски;
- не хотят подключаться к публичной Copy Trading-стратегии;
- готовы обсуждать отдельные условия покупки.

Контакт для индивидуального обсуждения: `@InvestExpert_info`.

## Инвестор с крупным депозитом

SolidarTrading можно использовать с депозитом разного размера.

Базовые правила для публичного Copy Trading:

- минимальный запуск: от `$100`;
- рекомендуемая базовая рабочая сумма: `$250`;
- при увеличении депозита удобно использовать суммы, кратные `$250`;
- крупный депозит не требует отдельного технического режима;
- резерв и поэтапное добавление средств могут быть разумнее одномоментного пополнения всей суммой.

Ограничение:

```text
Большой счет сам по себе не гарантирует меньший риск. При одинаковой относительной просадке убыток в деньгах на крупном счете будет больше.
```

## Кому SolidarTrading не подходит

Метод не подходит человеку, который:

- ждет гарантированную прибыль;
- не принимает возможность просадки;
- хочет использовать последние деньги;
- хочет использовать заемные средства;
- воспринимает Forex как банковский вклад;
- ожидает отсутствие рыночного риска;
- хочет бесконтрольно менять настройки;
- не готов соблюдать правила работы с балансом;
- хочет получить обещание фиксированной доходности;
- хочет автоматически переложить всю ответственность на робота или команду.

Короткий ответ:

```text
SolidarTrading - это торговая автоматизация, а не банковский вклад. Метод не подходит тем, кто не готов принять возможность просадки и рыночного риска.
```

## Что нельзя обещать

- SolidarTrading подходит всем без исключения.
- Опыт и понимание риска вообще не нужны.
- Крупный депозит автоматически безопасен.
- Увеличение депозита всегда уменьшает риск.
- Поэтапное пополнение гарантирует сохранность денег.
- Резерв гарантированно спасет счет.
- White Label предоставляется всем на одинаковых условиях.
- White Label запускается за фиксированный срок.
- Партнер может безопасно менять любые настройки.
- Цена робота фиксирована.
- Отдельная покупка гарантирует такой же результат, как Copy Trading.
- Другой брокер даст идентичное исполнение.
- Партнерский доход гарантирован.
- Крупный лидер автоматически становится частью команды.
