# 08_История проекта версии и изменения

## Назначение хаба

История SolidarTrading: ранний тестовый этап, сбой, рестарт, безопасный режим, смена версий, текущий `TiTan ATLAS`, правила старых видео и публичные границы по ранним убыткам.

Полные ответы хранятся в канонических карточках раздела `TiTan Bots - Метод и продвижение`. Этот файл хранит только карту вопросов и маршруты.
