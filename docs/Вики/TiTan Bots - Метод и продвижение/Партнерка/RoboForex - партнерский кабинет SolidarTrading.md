## Короткий ответ

Партнерский кабинет RoboForex позволяет активировать партнерскую программу, получить персональную ссылку, смотреть структуру, торговый оборот, начисления, использовать калькулятор и выводить партнерский доход.

Для сети SolidarTrading действует отдельная программа RoboForex `Special`. Это не стандартная `Infinity Partner Program`.

Быстрый путь:

```text
Личный кабинет RoboForex -> Partner -> Partner Dashboard
```

## Регистрация партнера

Новый пользователь должен регистрироваться по персональной ссылке своего спонсора.

Если спонсор есть:

1. получить у него персональную регистрационную ссылку RoboForex;
2. зарегистрироваться именно по этой ссылке;
3. пройти стандартную верификацию;
4. активировать партнерскую программу;
5. получить собственную персональную ссылку.

Если спонсора нет или пользователь его не знает, можно использовать общую ссылку проекта:

https://my.roboforex.com/en/?a=xvwag

Контакт основателя при отсутствии спонсора или проблемах с партнерским кабинетом:

https://t.me/InvestExpert_info

Не путать:

```text
персональная ссылка спонсора для регистрации -> верификация -> создание счета -> общая ссылка Copy Trading-стратегии SolidarTrading
```

## Активация партнерской программы

1. Войти в личный кабинет RoboForex.
2. Открыть `Partner`.
3. Выбрать `Partner Dashboard`.
4. Если программа еще не активирована, нажать `Стать партнером`.
5. После активации использовать `Partner Dashboard`.

Интерфейс RoboForex может меняться. Если пункты меню отличаются, ориентироваться на актуальный кабинет брокера.

## Что находится в Partner Dashboard

- `Partner account balance` - баланс партнерского счета;
- `Withdraw` - вывод средств;
- `Current program` - текущая программа;
- `Partner ID` - персональный идентификатор;
- `Partner link` - персональная регистрационная ссылка;
- текущий партнерский уровень;
- прогресс до следующего бонуса;
- инструменты привлечения клиентов.

## Где взять персональную ссылку

Основной путь:

```text
Partner -> Partner Dashboard -> Partner link
```

Дополнительный путь:

```text
Partner -> Links
```

Не давать новому пользователю общую ссылку проекта, если у него уже есть спонсор. Не путать регистрационную ссылку и ссылку подключения Copy Trading-стратегии.

## Где посмотреть структуру

Основные пути:

```text
Partner -> Partner Dashboard -> Your referrals
Partner -> Referrals
```

В разделе отображается дерево партнерской структуры: уровни, аккаунты, вложенные ветки и нижестоящие партнеры.

Состав отображаемых персональных данных зависит от настроек аккаунта, региона и правил брокера. Не сохранять в публичных материалах реальные имена, номера аккаунтов и необезличенные скриншоты структуры.

## Где посмотреть оборот и начисления

Основные разделы:

```text
Commission details
Commission
```

В кабинете можно увидеть данные, необходимые для расчета партнерского вознаграждения: торговую активность, закрытые ордера, оборот, уровень структуры и начисленную комиссию.

Начисления появляются после торговой активности и закрытых сделок. Регистрация или пополнение счета без торговли сами по себе не создают партнерское начисление.

## Где вывести партнерский доход

На странице `Partner Dashboard` в блоке `Partner account balance` нажать `Withdraw`.

Партнерский доход связан с отдельным партнерским счетом RoboForex. Подробные правила работы со средствами маршрутизируются в [03_Счет брокер деньги и вывод](../FAQ/03_%D0%A1%D1%87%D0%B5%D1%82%20%D0%B1%D1%80%D0%BE%D0%BA%D0%B5%D1%80%20%D0%B4%D0%B5%D0%BD%D1%8C%D0%B3%D0%B8%20%D0%B8%20%D0%B2%D1%8B%D0%B2%D0%BE%D0%B4.md).

## Партнерский калькулятор

Основные пути:

```text
Partner -> Partner Dashboard -> Partner calculator
Partner -> Calculators
```

Калькулятор помогает оценить возможные начисления по торговому обороту. Фактический результат зависит от реальной активности клиентов, инструментов, спреда и условий брокера.

Расчет калькулятора не является гарантией дохода.

Актуальные ставки по конкретным инструментам можно уточнить через партнерский отдел RoboForex:

https://my28.roboforex.org/en/help/add-a-ticket/

## Накопительная программа

У RoboForex есть дополнительная накопительная программа для партнеров с денежными бонусами и призами.

Основные пути:

```text
Partner -> Partner Dashboard -> Partner tiers
Partner -> Loyalty program
```

Уровни, пороги и призы могут меняться. Актуальные условия нужно смотреть в личном кабинете. Не обещать конкретный приз без проверки текущего кабинета.
