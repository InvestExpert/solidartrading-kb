## Короткий ответ

Фиксированной платы за подключение к SolidarTrading нет.

Сейчас действует льготная комиссия `10%` от новой прибыли, полученной через RoboForex Copy Trading. Комиссия начисляется автоматически только при положительном результате. Если новой прибыли нет, комиссия не списывается.

Короткая продажная формулировка:

```text
Вход без фиксированной платы. Команда получает вознаграждение только тогда, когда система создает вам новую прибыль.
```

## История условий

- Раннее условие тестового периода: `5%` от прибыли.
- Актуальное условие по подтвержденному raw-файлу от `2026-06-01`: `10%` от прибыли.
- Комиссия может измениться в будущем по мере развития системы и накопления статистики.
- На рынке Copy Trading встречаются ставки от `5%` до `50%`, поэтому текущие `10%` можно подавать как конкурентное условие.

Не писать просто:

```text
Подключение полностью бесплатное.
```

Правильно:

```text
Фиксированной платы за подключение нет. Команда получает 10% только от новой прибыли, которую система заработала пользователю.
```

## Как рассчитывается Performance Fee

Для SolidarTrading используется тип комиссии `Performance Fee`: процент от результата инвестора, полученного при копировании сделок.

По официальной логике RoboForex расчет идет по общему результату инвестора за период, а не по каждой отдельной прибыльной сделке.

В расчете важно:

- учитывается совокупный результат сделок периода;
- учитываются открытые и закрытые позиции;
- из результата вычитаются торговые расходы, например swap и ECN/Prime-комиссии;
- комиссия начисляется только с новой прибыли;
- действует принцип `High Watermark`;
- расчет выполняется в конце инвестиционного периода или при отписке инвестора.

Исправление старой гипотезы:

```text
Комиссия не списывается с каждой отдельной прибыльной закрытой сделки.
```

Точная версия:

```text
Комиссия рассчитывается по общему результату инвестиционного периода. Система учитывает совокупный результат сделок, вычитает торговые расходы и начисляет 10% только с новой чистой прибыли.
```

## High Watermark простым языком

`High Watermark` означает, что комиссия берется только с нового максимума прибыли.

Пользователь не платит повторно:

- за восстановление после просадки;
- за возврат к уже достигнутому уровню;
- за колебания счета без нового максимума;
- за убыточный инвестиционный период.

Простой пример:

```text
Неделя 1:
новая прибыль = $100
комиссия 10% = $10

Неделя 2:
общая прибыль выросла до $150
новая прибыль = $50
комиссия 10% = $5

Неделя 3:
результат снизился до $140
новой прибыли нет
комиссия = $0
```

Продажная формулировка:

```text
Комиссия устроена честно: мастер-счет получает вознаграждение только тогда, когда создает для пользователя новый финансовый результат. За восстановление старой просадки повторно платить не нужно.
```

## Когда списывается комиссия

Текущий импорт фиксирует инвестиционный период `1 неделя`.

Комиссия:

- резервируется на счете инвестора;
- не переводится мастер-счету сразу после отдельной сделки;
- перечисляется мастер-счету в конце инвестиционного периода;
- также рассчитывается при отмене подписки инвестором.

Короткий ответ:

```text
Комиссия рассчитывается раз в неделю и списывается автоматически только при появлении новой чистой прибыли. Если пользователь отменяет подписку, итоговый расчет выполняется при отключении.
```

## Кто получает комиссию

Комиссия перечисляется мастер-счету стратегии автоматически через инфраструктуру RoboForex Copy Trading.

Для новичка не нужно перегружать техническими деталями распределения.

Допустимая формулировка:

```text
Команда зарабатывает только тогда, когда робот создает новую прибыль пользователю.
```

## Какие еще расходы есть

Нужно отделять три разных понятия:

1. Комиссия SolidarTrading:
   - `10%` от новой прибыли;
   - выплачивается мастер-счету через Copy Trading.

2. Брокерские торговые расходы:
   - spread;
   - swap;
   - комиссии брокера по типу счета;
   - учитываются в реальном торговом результате.

3. Партнерские выплаты:
   - относятся к экономике брокера и торгового оборота;
   - не являются комиссией с депозита клиента;
   - подробно раскрываются в хабе [07_Партнерская программа](../FAQ/07_%D0%9F%D0%B0%D1%80%D1%82%D0%BD%D0%B5%D1%80%D1%81%D0%BA%D0%B0%D1%8F%20%D0%BF%D1%80%D0%BE%D0%B3%D1%80%D0%B0%D0%BC%D0%BC%D0%B0.md).

Короткая формулировка:

```text
Помимо 10% Performance Fee, в торговле присутствуют стандартные брокерские расходы. Они являются обычной частью работы на финансовом рынке и учитываются в фактическом результате счета.
```

## Что нельзя обещать

- что комиссия всегда останется `10%`;
- что условия RoboForex не изменятся;
- что брокерские расходы отсутствуют;
- что комиссия гарантирует прибыль;
- что подключение "полностью бесплатно" без пояснения Performance Fee.
