## Короткий ответ

Солидарная десятка - это командная цепочка из десяти позиций, которая помогает участникам использовать глубину десятиуровневой партнерской программы SolidarTrading.

Механика разработана, проверялась на раннем этапе и использовалась в нескольких тестовых десятках. Новые мини-команды формируются по заявкам участников через лидеров Solidar.

## Структура

| Позиция | Роль |
| ---: | --- |
| 1 | верхний солидарный аккаунт клуба Solidar |
| 2 | лидер десятки |
| 3-9 | семь участников |
| 10 | нижний замыкающий аккаунт с R-роботом |

Формула:

```text
1 верхний аккаунт Solidar + 1 лидер + 7 участников + 1 нижний аккаунт с роботом = 10 позиций
```

## Что база знает

- солидарные десятки являются подтвержденной командной механикой, а не только ранней гипотезой;
- механика использует десять позиций расширенной партнерской программы;
- нижний аккаунт замыкает структуру и использует R-робота;
- верхний аккаунт принадлежит клубу Solidar;
- лидер формирует цепочку, объясняет правила участия и управляет нижним аккаунтом;
- нижний замыкающий аккаунт принадлежит лидеру десятки;
- базовый формат финансирования нижнего аккаунта - общий пул участников;
- партнерские начисления автоматически отображаются в личных кабинетах RoboForex и не распределяются вручную из общего пула;
- новая десятка запускается после формирования полной цепочки;
- участие в солидарной десятке не является обязательным условием SolidarTrading.

## Роль лидера

До подключения лидер должен объяснить:

- размер участия;
- порядок фиксации взносов;
- правила управления общим пулом;
- порядок использования торговой прибыли;
- порядок распределения возможного убытка;
- условия выхода и возврата средств при закрытии десятки.

Лидер формирует цепочку участников вручную и при необходимости помогает скорректировать структуру по правилам RoboForex.

## Партнерские начисления без самостоятельных приглашений

Солидарная десятка позволяет участнику получать партнерские начисления без самостоятельного приглашения людей.

После формирования десятки участник получает место в цепочке, минимум один нижестоящий уровень и источник партнерских начислений после запуска нижнего аккаунта.

Это не гарантия фиксированной суммы, срока окупаемости или неизменного торгового оборота.

## Как вступить

1. Обратиться к своему лидеру или спонсору.
2. Уточнить, формирует ли он солидарную десятку.
3. Ознакомиться с правилами участия.
4. Если подходящего лидера нет, написать в общий чат Solidar.
5. Выбрать подходящий вариант из доступных лидеров.

## Ограничения

Солидарная десятка:

- не гарантирует фиксированный доход;
- не гарантирует одинаковые начисления каждому участнику;
- не отменяет необходимость торгового оборота;
- не превращает партнерскую программу в выплату за регистрацию;
- не является обещанием фиксированного срока окупаемости;
- не отменяет торговый риск нижнего аккаунта;
- не гарантирует возврат средств в любой момент вне заранее согласованных правил пула.
