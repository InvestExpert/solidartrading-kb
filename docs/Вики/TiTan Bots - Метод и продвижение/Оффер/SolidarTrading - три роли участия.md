## Суть

Презентация `Где прячется доход` предлагает три роли участия. Это удобно для продажи, потому что один и тот же метод можно объяснять разным людям по-разному.

По хабу 10 эти роли нужно объяснять как разные маршруты, а не как один смешанный оффер. Новичку сначала показывается основной публичный формат Copy Trading. White Label, отдельная покупка TiTan ATLAS и индивидуальные условия раскрываются только при соответствующем запросе.

## 1. User

Для человека, который хочет использовать систему для своего счета.

Рабочий смысл:

- деньги остаются на личном счете;
- торговля автоматизируется;
- человек не тратит время на ручную торговлю;
- контроль вывода остается у клиента.

Безопасная формулировка:

```text
Вы подключаете автоматизацию к своему счету и сохраняете контроль над деньгами. Результат зависит от рынка, настроек и выбранного риска.
```

## 2. Partner

Для лидера команды, блогера, владельца аудитории или человека, который хочет строить партнерскую модель.

Рабочий смысл:

- партнер продвигает метод;
- доход строится вокруг активности клиентов и оборота;
- модель должна быть привязана к реальному использованию брокерской инфраструктуры.
- собственный торговый депозит для партнерского направления не обязателен;
- лидер - это партнер, который строит, сопровождает и развивает команду.

Безопасная формулировка:

```text
Партнер может развивать структуру и получать вознаграждение от торгового оборота. Собственный торговый депозит для партнерского направления не обязателен.
```

## 3. Business / White Label

Для компаний, команд и проектов, которые хотят запускать собственную упаковку на базе технологий SolidarTrading.

Рабочий смысл:

- свой бренд;
- техническая основа на стороне SolidarTrading;
- отдельная упаковка для аудитории партнера.
- отдельный мастер-счет;
- Copy Trading-стратегия;
- партнерская структура;
- настройки под задачи проекта;
- обучение и сопровождение по договору.

Статус: `available_by_individual_contract`.

White Label уже доступен для лидеров, команд и компаний по индивидуальному договору. Не обещать фиксированный срок, цену, пакет услуг или передачу исходного кода.

Безопасная формулировка:

```text
Для команд и компаний возможен White Label: отдельный продукт под своим брендом на базе технологии TiTan Bots. Формат, сроки, настройки и сопровождение обсуждаются индивидуально.
```

## Что не показывать первым ответом новичку

- White Label;
- отдельную покупку TiTan ATLAS;
- индивидуальные условия лидера;
- крупный депозит;
- передачу настроек;
- другого брокера.

Первый ответ новичку:

```text
Основной формат SolidarTrading - подключение TiTan ATLAS через Copy Trading на личном брокерском счете. Вам не нужно покупать робота, устанавливать советник и самостоятельно торговать.
```
