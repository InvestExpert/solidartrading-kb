## Короткий ответ

TiTan Bots надо объяснять не как один торговый файл, а как продуктовую систему вокруг роботов, тестов, контроля риска, сопровождения и вариантов монетизации.

Сильная рамка:

```text
TiTan Bots - это торговая автоматизация, которую можно использовать как метод для клиента, продукт для партнера и технологическую основу для команды или компании.
```

## Что база знает

- Для текущего публичного направления SolidarTrading основной робот - [TiTan ATLAS в линейке TiTan Bots](../%D0%9C%D0%B5%D1%82%D0%BE%D0%B4/TiTan%20Bots%20-%20%D0%BB%D0%B8%D0%BD%D0%B5%D0%B9%D0%BA%D0%B0%20%D1%82%D0%BE%D1%80%D0%B3%D0%BE%D0%B2%D1%8B%D1%85%20%D1%80%D0%BE%D0%B1%D0%BE%D1%82%D0%BE%D0%B2%20%D0%B4%D0%BB%D1%8F%20SolidarTrading.md).
- Для обычного пользователя главная выгода - торговая автоматизация на своем счете, без передачи денег в чужое управление.
- Для партнера главная выгода - готовая тема для продвижения и модель вокруг реального торгового оборота.
- Для команды или компании White Label уже доступен по индивидуальному договору. Это не массовый тариф, а отдельный формат под задачи клиента.
- Для инвестора или предпринимателя TiTan Bots можно показывать как продуктовую модель вокруг финансовых рынков, а не как разовую продажу советника.
- Отдельную версию TiTan ATLAS можно купить индивидуально как защищенный файл без исходного кода. Пакет, цена, обновления и сопровождение определяются договором.

## Варианты участия

### Пользователь

Смысл:

- подключить автоматизацию к своему счету;
- не торговать руками;
- сохранить контроль над деньгами;
- понимать, что рынок и просадка остаются.

Продажная формулировка:

```text
Вы не отдаете деньги проекту. Деньги остаются на вашем счете, а торговую работу берет на себя система.
```

### Партнер или лидер

Смысл:

- продвигать продукт TiTan Bots / SolidarTrading;
- получать вознаграждение по партнерской модели;
- не разрабатывать роботов с нуля;
- вести людей в понятную систему, а не в обещание выплат из воздуха.

Продажная формулировка:

```text
Партнер продвигает не обещание, а рабочую торговую инфраструктуру: счет, робот, копирование, оборот и прозрачную модель.
```

### Business / White Label

Смысл:

- отдельная упаковка под бренд команды или компании;
- технология TiTan Bots внутри;
- отдельный мастер-счет партнера или проекта;
- Copy Trading-стратегия;
- партнерская структура;
- настройки под задачи проекта;
- обучение и сопровождение;
- возможность запускать продукт для своей аудитории;
- условия, ответственность, контроль настроек и поддержка согласуются отдельно.

Продажная формулировка:

```text
Команда может получить не идею с нуля, а готовую технологическую основу для собственного финансового продукта.
```

Граница:

```text
White Label уже доступен для лидеров, команд и компаний по индивидуальному договору. Исходный код в стандартном White Label не передается.
```

### Покупка или управляемый формат

Доступны два индивидуальных направления:

- отдельная покупка TiTan ATLAS для самостоятельной работы;
- управляемый вариант, где команда TiTan Bots помогает с настройками, пресетами, обновлениями и сопровождением.

Это не основной массовый оффер SolidarTrading. Основной публичный формат для обычного пользователя остается Copy Trading.

При отдельной покупке пользователь может получить лицензию, защищенный файл робота без исходного кода, установку, проверку совместимости с брокером, рекомендуемый пресет, помощь с настройкой, обучение, сопровождение и обновления. Конкретный пакет и цена обсуждаются индивидуально с создателем проекта: `@InvestExpert_info`.

## White Label и отдельная покупка - разные форматы

```text
Отдельная покупка - это защищенная версия робота для самостоятельной работы.
White Label - запуск отдельного брендированного продукта с мастер-счетом, Copy Trading и партнерской моделью.
```

## Другой брокер

Отдельную версию TiTan ATLAS можно использовать у другого брокера после проверки совместимости торговой платформы, типа счета, инструментов, комиссий, плеча, исполнения и ограничений брокера.

Не обещать одинаковый результат у разных брокеров.

## Настройки и ответственность

В отдельной покупке и White Label настройки могут быть доступны в рамках договора.

Граница:

```text
Бесконтрольные изменения не рекомендуются: они могут увеличить риск и сделать результат несопоставимым с публичной стратегией.
```

## Ключевой принцип модели

```text
Пользователь зарабатывает -> компания зарабатывает.
```

Смысл не в том, чтобы взять деньги клиента в управление, а в том, чтобы построить модель вокруг долгой работы счета, оборота, результата и доверия.

## Ограничения

- Не обещать гарантированную доходность.
- Не продавать TiTan Bots как файл, который можно безопасно поставить без сопровождения.
- Не обещать White Label за фиксированный срок или фиксированную цену.
- Не смешивать покупку продукта, копитрейдинг, партнерку и White Label в один оффер.
- Не использовать тесты как гарантию будущего результата.
- Не обещать передачу исходного кода.
- Не обещать работу у любого брокера без проверки совместимости.
- Не обещать повторение публичной статистики после самостоятельных изменений.
