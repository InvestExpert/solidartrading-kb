# TiTan Bots - линейка торговых роботов для SolidarTrading

## Короткий ответ

TiTan Bots - это не один случайный робот, а линейка торговых систем. Для текущего публичного направления SolidarTrading основной робот - [[../../../TiTan Atlas (BA)|TiTan ATLAS]].

Продажная формулировка:

```text
Сейчас в SolidarTrading главный робот - ATLAS. Это фундамент системы: вокруг него уже есть тесты, мониторинг, защита и понятная модель подключения.
```

## Что база знает

- В технической базе уже есть отдельные страницы по [[../../../TiTan Atlas (BA)|TiTan ATLAS]], [[../../../TiTan Hyperion (FLAT)|TiTan HYPERION]], [[../../../TiTan Helios (Bollinger ADX)|TiTan HELIOS]] и [[../../../TiTan Kronos (CE)|TiTan KRONOS]].
- Для клиента не нужно начинать с внутренней архитектуры и кода.
- В публичном SolidarTrading сейчас фокус на ATLAS.
- Остальные роботы можно описывать как более широкую линейку TiTan Bots, которая будет развиваться постепенно.
- Для обычного пользователя основной сценарий - подключиться к системе, а не покупать робота как коробочную программу.

## Факты

- ATLAS - текущий главный робот публичного направления SolidarTrading.
- HYPERION связан с FLAT-логикой и гибридным балансом.
- HELIOS связан с Bollinger/ADX и направлением тренда.
- KRONOS связан с Chandler Exit, контролем выхода и завершением цикла.
- Названия и роли роботов зафиксированы в техническом разделе TiTan Bots.

## Как объяснять клиенту

Коротко:

```text
У нас не один робот "на все случаи жизни", а линейка TiTan Bots. Но мы не запускаем все сразу. Сейчас главный публичный робот - ATLAS, потому что по нему есть основная проверка, мониторинг и понятная модель подключения.
```

Если спрашивают, можно ли купить робота:

```text
В текущем публичном направлении роботы не продаются как обычная программа. Основной формат - подключение к системе через SolidarTrading.
```

Если спрашивают, какие роботы есть:

```text
В линейке TiTan Bots есть ATLAS, HYPERION, HELIOS и KRONOS. Сейчас для SolidarTrading основной робот - ATLAS. Остальные относятся к более широкой линейке и могут запускаться постепенно.
```

## Ограничения

- Не говорить, что все роботы уже доступны для публичного подключения.
- Не обещать доходность по HYPERION, HELIOS или KRONOS без отдельной проверки.
- Не продавать ATLAS как робот без риска.
- Не смешивать клиентское объяснение SolidarTrading с внутренней технической документацией.
