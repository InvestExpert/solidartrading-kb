# Метод - Индекс

Раздел слоя [[../TiTan Bots - Метод и продвижение - Индекс|TiTan Bots - Метод и продвижение]].

## Что сюда класть

- простое объяснение метода TiTan Bots;
- копитрейдинг как способ доступа к работе ботов;
- логика "что получает человек" без технических деталей кода;
- сравнение метода с ручной торговлей и обычными инвестициями;
- ограничения метода, если они важны для честного объяснения.
