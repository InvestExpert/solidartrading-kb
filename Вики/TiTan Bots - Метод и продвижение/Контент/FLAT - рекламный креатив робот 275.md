# FLAT - рекламный креатив робот 275

## Суть

Рекламный креатив для [[../../../TiTan Hyperion (FLAT)|TiTan Hyperion (FLAT)]] с роботом, графиком и FOMO-оффером `$275`.

Файл актива:

![FLAT - рекламный креатив робот 275](./Assets/FLAT%20-%20%D1%80%D0%B5%D0%BA%D0%BB%D0%B0%D0%BC%D0%BD%D1%8B%D0%B9%20%D0%BA%D1%80%D0%B5%D0%B0%D1%82%D0%B8%D0%B2%20%D1%80%D0%BE%D0%B1%D0%BE%D1%82%20275.png)

## Применимо к

- FLAT;
- рекламный пост;
- Telegram;
- прогрев;
- запуск оффера.

## Источник

`Flat.png`

## Текст на креативе

```text
FLAT - бот, за который платят, чтобы им просто владеть

$275 только до 01.09.
Потом? Готовь $1000+

✓ Проходит челенджи
✓ Выводит прибыль
Там, где другие сливают

Бери сейчас - или потом плати в 4 раза больше
```

## Сильные стороны

- понятный FOMO;
- крупный заголовок;
- образ торгового робота;
- визуальная связь с рынком через свечной график;
- сразу показывает цену и дедлайн.

## Риски

- утверждение "проходит челенджи" требует доказательства;
- утверждение "выводит прибыль" может звучать как обещание дохода;
- "там, где другие сливают" требует осторожной подачи;
- дата `01.09` без года не годится для долгосрочной базы;
- `$1000+` требует источника и актуальности.

## Решение по статусу

Креатив сохранить как черновик и исходник рекламной идеи.

Перед публикацией:

- проверить кейсы;
- добавить год к дедлайну;
- смягчить финансовые обещания;
- добавить дисклеймер;
- проверить, не нарушает ли текст правила площадки.
