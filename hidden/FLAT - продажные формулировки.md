## Позиционирование

FLAT нужно объяснять не техническими терминами, а через понятную выгоду.

Главная мысль:

```text
FLAT - бот для тех, кто не хочет угадывать рынок вручную.
```

Простое объяснение:

- он не спрашивает: "куда пойдет цена?";
- он работает по заранее заданным правилам;
- он старается использовать само движение цены;
- для клиента смысл простой: меньше ручной торговли, больше автоматизации.

Если говорить сильнее:

```text
Пока одни пытаются угадать идеальный вход, FLAT работает по системе.
```

Для челенджей и проп-компаний это можно подавать отдельно, но только при наличии доказанных кейсов.

## Формулировки

### FLAT - новый класс торгового бота

Использование:

- презентация;
- лендинг;
- видео.

Смысл: FLAT подается не как обычная сетка и не как бот, который угадывает направление.

Человеческая версия:

```text
FLAT - это не бот-гадалка.

Он не пытается предсказать рынок.
Он работает по правилам и старается забирать движение там, где человек часто сомневается.
```

Риск: нужно объяснять, чем именно он отличается от обычной сетки.

### FLAT не гадает направление - FLAT монетизирует рынок

Использование:

- лендинг;
- видео;
- презентация.

Смысл: бот не пытается предсказать направление, а работает с движением и диапазоном.

Человеческая версия:

```text
Тебе не нужно сидеть у графика и гадать, куда сейчас пойдет цена.

Идея FLAT проще: рынок двигается - бот работает по своим правилам.
```

Риск: формулировка сильная, но требует технического объяснения на странице метода.

### Берем деньги там, где другим скучно, и там, где другим больно

Использование:

- branding;
- посты;
- видео.

Смысл: FLAT должен работать и в спокойном рынке, и в резком движении.

Человеческая версия:

```text
Когда рынок стоит - большинство скучает.
Когда рынок дергается - большинство нервничает.

FLAT создан, чтобы работать и там, и там.
```

Риск: звучит как обещание результата. Нужен дисклеймер и доказательства.

### FLAT - бот, за который платят, чтобы им просто владеть

Использование:

- FOMO-маркетинг;
- Telegram;
- лендинг.

Смысл: усиление ценности владения ботом.

Человеческая версия:

```text
Это не сигнал в чат и не совет "купи-продай".

Это автоматизация, которая работает на счете по заданным правилам.
```

Риск: формулировка агрессивная, требует доказательств спроса и осторожной подачи.

### $275 только до 01.09. Потом? Готовь $1000+

Использование:

- запуск;
- ограниченный оффер;
- Telegram.

Человеческая версия:

```text
Сейчас вход дешевле.
Если результат и спрос подтвердятся, цена может быть другой.
```

Риск: обязательно указать год, источник, актуальность и условия. Без этого не использовать как готовый оффер.

### Проходит челенджи и зарабатывает

Использование:

- prop audience;
- кейсы;
- прогрев.

Человеческая версия:

```text
Если есть подтвержденные кейсы, FLAT можно показывать как инструмент для челенджей.

Без кейсов это только гипотеза для проверки, а не обещание.
```

Риск: использовать только при наличии подтвержденных кейсов. Иначе заменить на более осторожную формулировку.

## Правило для этой страницы

Не писать для клиента:

- `anti-guessing system`;
- `hybrid grid + hedge`;
- `fork logic`;
- `market monetization engine`;
- `prop challenge bot`.

Если эти идеи нужны, переводить их простым языком:

- не угадывает рынок;
- работает по правилам;
- автоматизирует торговлю;
- старается использовать движение цены;
- может быть полезен для челенджей только при наличии доказанных кейсов.
